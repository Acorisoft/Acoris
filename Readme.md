# 风语者之歌

风语者之歌，写给那个勇敢的你！也写给那个天真浪漫的你！也写给那个曾经年少的你。

## 旋律

风语者之歌是分为三个旋律的，每个旋律都有自己的核心内容。每个旋律发生的故事都不一样，发生的地点也不一样，但是每个旋律之间的故事都不是独立的，每个故事之间都会有一些互动。在风语者之歌中，这三个旋律分别是：成长、浪漫、回忆。

### 成长

风语者之歌在这个旋律下，它讲述的故事主要是故事中的人物成长的故事，他更多的是关注于人成长这件事情本身。他讲述的故事主要发生在架空的世界里，里面会有剑与魔法也会有骑士与巫女，也会有赛博朋克等等元素。

>  之所以风语者之歌会选择使用架空而不是使用青春校园来表达成长，那是因为现实中一个人的成长，对于一个人而言可能是不会带来任何价值的。一个人的成长是经历了许多挫折和矛盾最终成就而成的，每个人的成长对于自己而言是一段漫长的过程，而对于他人而言这就像是手机应用的一条推送，比如微博的一条热搜，引起不了多少兴趣。你用多久的时间才能成就自己，而你的成就在别人眼前也只是转瞬即逝的烟火。我更想让那些获得成长的人能够获得瞩目的焦点，让那些平凡的人能够拥有属于自己的掌声，拥有别人送上来的由衷的祝福。所以我选择了使用架空世界，因为现实世界的故事，也意味着故事必须现实，我想摆脱这种束缚，让这个理想能够在另一个世界中得到实现。当然肯定会有人质疑，这种屁大点成就就成为他人瞩目的焦点，那不是自我感动？确实在他们眼里看上去只剩下自我感动，但是我想反问一下，凭什么那些屁大点事就占着别人热搜的人配拥有万众的瞩目，而你就不可以？



> 草根文化，不代表着低俗，但是也不能摆脱低俗。真正的草根文化如果想要做好，就要思考草根文化本质上的述求，那就是表达自我。当然自我的表达局限于文化与眼界，为了真正保护每个人的差异，风语者之歌里面不会选择任何先天决定造成差异化的元素来表达自我。

关键字: 热血、勇敢、燃

### 浪漫

风语者之歌的第二个旋律，叫做浪漫。这个旋律下的故事主要是讲述情窦初开的少男少女在异世界中，发现自我、接纳自我、承认自我、拥抱自我的故事。

> 可能你已经注意到了，这个世界的主题和第一个旋律还是有点相似的。第一个主题主要是成长，而第二个主要是发现自我、接纳自我的过程。其实本质上也是成长，这个故事主要是写给那些因为自身原因不能拥抱他人的人，这些人是：那些内心经历过非常人能及痛苦的人、那些社恐或者因为各种生理、心理创伤或者病痛无妨拥抱他人的人。
>
> 这个故事，是关注那些弱势、敏感、脆弱的群体。他们虽然是弱势的群体，那只是指他们的内心，但实际上他们可能是比我们更有天赋的人。这个旋律下面最主要是少男少女之间的故事，我会用尽量朦胧的笔触去刻画每个人的性格，但是千万要注意，这里的浪漫并不止有爱情。不要过度恋爱脑来看待这里面每一个角色的互动。（让更多的cp成为可能？）

关键字：唯美、浪漫、朦胧

### 回忆

风语者之歌的最后一个旋律，就是回忆了。这个旋律之下讲述的是重逢的故事。也许是热血也许是浪漫，但最终是需要面对现实。这个旋律偏向于角色们在穿越之后最终回到现实之后，每个人如何学会将其他旋律中学到的东西运用到现实的故事。当然也可能会有其他的篇章。情怀用的。

## 机制

也许你已经看腻了前面的介绍，想要了解一下风语者之歌这个世界观里面包含的内容。那么这里就开始介绍了。

### 门

门是风语者之歌中最重要的元素！我们需要通过门连通现实世界与异世界的大门。在风语者之歌中，它原本的名字就叫做门，但是因为门上刻有四只象作为纹理，所以很多人都喜欢把它叫做四象门。四象门代表着风语者之歌中的穿越道具，它也有自己的规则。

#### 穿越规则

>  首先来了解一下穿越的规则有哪些？
>
> 1. 穿越方式
> 2. 穿越限制
> 3. 时间规则
> 4. 穿越地点
> 5. 穿越开启时间
>
> 这是穿越规则中最主要的几个规则，第一个规则穿越方式其实也很好理解，就是投胎还是身体穿越

四象门基本上都是身体穿越，穿越过程中不允许携带东西到另外的世界。四象门的穿越地点和时间规则还有开启时间与穿越的目标世界有关。

#### 资格规则

通过四象门的人，都需要资格验证。只有通过资格验证、拥有通行资格的人才能通过四象门往返两个世界。

### 能力

能力是风语者之歌的另外一个元素，通过四象门穿越到异世界之后，就能够解锁能力了。能力其实就是法术、魔术、巫术、咒术等一系列超能力的统称，在风语者之歌中它们都叫能力，不同能力都应用着同一套规则。

#### 能力的获得

能力需要通过收集能力碎片获得，一般来说收集能力碎片的场地在厄瑞克斯的每一处遗迹，遗迹中保存着历代知名的风语者的遗产。

#### 能力的发动

能力的发动需要划分类型，能力分为四种：被动、主动、陷阱、诅咒。

* 对于被动能力的发动，一般而言是不需要代价的，并且是一种天赋。
* 对于主动能力的发动，需要满足能力的冷却时间已经结束并且支付相应的代价。
* 对于陷阱能力的发动，第一次发动需要耗费代价，第二次发动需要满足陷阱发动的条件。
* 对于诅咒能力的发动，需要消耗诅咒的媒介。



#### 能力发动的代价

* 能力的发动需要代价，代价有两种：
  * 代币
  * 特殊的仪式
* 能力所消耗的代币又能力决定
* 有些能力在没有代币的情况下可以通过特殊的仪式来发动

#### 能力与权能

权能是能力的升级版，可以获得比普通能力更强的效果，又或者消耗的代价比能力要小很多。总而言之权能非常稀有。

#### 权能与继承

权能和能力不一样，权能可以让人继承，但是能力不行。

## 异世界

在成长、浪漫这两大个旋律中，都会涉及到异世界这个概念。在成长的旋律中，主要的世界叫做厄瑞克斯；而在浪漫这个旋律中，主要的世界叫做遗忘之都。

### 厄瑞克斯

厄瑞克斯是成长旋律中最主要的冒险舞台，在这个世界里你将会为了成为风语者而努力，风语者是遗忘之都的守护者。

#### 厄瑞克斯的时间流速

厄瑞克斯的世界中，时间流速比现实世界慢非常多，现实世界中过去的一分钟，在厄瑞克斯的世界里时间只过去了两秒。

#### 厄瑞克斯的通行资格

通往厄瑞克斯的世界，必须满足下面任意一条条件：

* 初一、初二的同学，包括休学的同学
* 高一、高二的同学，包括休学的同学
* 大一、大二、大三的同学，包括休学的同学
* 接受并通过厄瑞克斯的试炼的人

#### 厄瑞克斯的开启时间

* 每个假期或者周末

#### 厄瑞克斯的NPC

* 掌管厄瑞克斯四象门的守门人
* 见证厄瑞克斯世界中所有历史的
* 待补充

### 遗忘之都

遗忘之都是浪漫旋律中最主要的冒险舞台，在遗忘之都里面会出现不同的意象，每个人都有符合自己意象的地方，也有特殊的意象是专属意象。

#### 意象

意象是每个遗忘之都中的人都拥有的抽象，是它们内心的映射。

#### 守护者/兽

守护者或者守护兽是它们专属的保护吉祥物，用来保护他们的安全。

> 守护者或者守护兽有两种可能，一种是厄瑞克斯世界的风语者的映射。又或者是作为与厄瑞克斯世界的风语者之间沟通的中间人

## 仪式

风语者之歌的故事，发生在这座人口仅有200万人的小城市里，这座城市虽然依山傍水但是经济不怎么发达，虽然经济不发达但是这座城市里的学校文娱活动确实令人惊羡。

这里的学校，每年都会展开两次文艺晚会，别看数量不多，但是十校联谊却非常让人心动。其实明面上是演出，但实际上是这座城市特有的学生会招新。

老师每年都会接到推荐每班至少10名新生的指标，目的就是为了那些有“问题”的学生通过加入学生会来获得成长。

### 征召仪式

征召仪式主要针对初高中还有大学新生。他们有意愿加入学生会的就会举行征召仪式，征召仪式的主要内容就是每个人依次走进联通着哲理之海的四象门并从里面出来。在这个短暂的过程中，哲理之海就会针对每个人的特质，赋予他们不同四象门的同行资格。

#### 征召仪式的第一个议程

[征召仪式](\征召仪式.md)的第一个议程是抉择。每个人都需要选择一个代表自己的主题。

#### 征召仪式的第二个议程

征召仪式的第二个议程是许愿。每个人都需要选择一个自己喜欢的物品，并且告知主持，这个物品将会送给谁。

### 退选仪式

退选仪式实际上是针对那些已经不再符合通行资格的人举办的，他们将会在门内留下他们的精神遗产，留作后人所用。

#### 退选仪式的精神遗产

退选的最后遗留下来的遗产有好几种：

* 权能碎片
* 能力碎片
* 祝福碎片

在征召期间获得成就越高，留下的遗产就越强，最好的成就将会留下权能，其次会留下能力碎片，最次也会留下祝福碎片。每个人留下的遗产都会散落在世界各处，成就越高，越不分散。极其所有碎片能够拼凑出留下这个遗产的人的信息。

> 一般来说权能最好，能力其次，祝福最差。但是不要笃信权能所给人带来的加持，往往能够帮助你的更是那些无名的祝福。

### 献祭仪式

一种失传很久的仪式，只有在极其危险的情况下才会允许献祭仪式举行。通过献祭仪式能够手动造神，造出的神祇将失去人的资格。
